Terminologias:

a) Definição: é um enunciado que caracteriza um determinado termo.
Ex:
* Um número inteiro $a$ é dito par se $a = 2k$, $k \in \mathbb{Z}$
* Um número inteiro $a$ é dito impa se $a = 2k 1$, $k \in \mathbb{Z}$

b) Axioma ou postulado: é uma sentença que não é provada e é considerada como óbvia.
Ex:
* Axiomas de Peano
* Postulados de Euclides

c) Proposição: é uma sentença declarativa que, quando verdadeira deve ser provada.
Ex:
* A soma de dois números pares é um número par.
Ou em linguagem maremática
$m, n \in \mathbb{Z}$ pares $\rightarrow$ é par.

d) Teorema: é uma proposição importante.
Ex:
* Teorema de Pitágoras

e) Corolário: é uma consequência direta de um teorema.

f) Lema: é um pré-teorema, ou seja, é um resultado que ajuda a demonstração de um teorema.
