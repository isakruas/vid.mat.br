Demonstrações
P $\rightarrow$ Q
Lê-se:
* P implica Q
* P é hipótese
* Q é a tese
c * Se P então Q
* P é condição suficiente para Q
* Q é condição necessária para P
a) Demonstração direta
P $\rightarrow$ Q
P $\rightarrow$ ... $\rightarrow$ Q
Ex:
A soma de dois números pares é um numero par
Def:
$a$ $\in \mathbb{Z}$ é par se $a = 2k$, $ k \in \mathbb{Z}$
Prova:
Hipótese: $m,n \in \mathbb{Z}$ são pares.
Tese: $m n$ é par.
Demonstração: $m, n$ são pares $\rightarrow m = 2k$ e $n = 2q$
$k, q \in \mathbb{Z} \rightarrow m n = 2k 2q = 2(k q)$
$k q \in \mathbb{Z} \rightarrow m n$ é par.
Ex. 2
Se $n$ é um número impa, então $n^2$ também é impa.
Def. $a \in \mathbb{Z}$ é ímpa se $a = 2k 1, k \in \mathbb{Z}$
Prova:
$n$ é ímpa $\rightarrow n = 2k 1, k \in \mathbb{Z}$
$\rightarrow n^2 = (2k 1)^2 = 4k^2 4k 1 = 2(2k^2 2k) 1$
$2k^2 2k \in \mathbb{Z}$
$\rightarrow n^2$ é impar
b) Demonstração por contraposição
P $\rightarrow$ Q
~P $\rightarrow$ ~Q
Ex.
Se $n^2$ é par, então $n$ também é par.
P = $n^2$ é par
Q = também é par
~Q = $n$ é impa
~P = $n^2$ é impa
Ex.2
Seja $n$ um número inteiro. Prove que se $3n 2$ é ímpa, então $n$ é ímpa.
P = $3n 2$ é ímpa
Q = $n$ é ímpa
Prova:
Se $n$ é par $\rightarrow$ $n = 2k, k \in \mathbb{Z}$
Daí,
$3n 2 = 3(2k) 2 = 2(3k 1)$
$3k 1 \in \mathbb{Z}$
$\rightarrow 3n 2$ é par
c) Demonstração por contradição ou absurdo
P $\rightarrow$ Q
Assume que P acontece e vai supor que o Q não aconteça
Ex.
$\sqrt{2}$ é irracional.
Hipótese: toda a bagagem matemática
Tese: $\sqrt{2}$ é irracional
Pova:
Suponha que $\sqrt{2}$ é racional
$\sqrt{2} = \frac{a}{b}, a, b, b = 0, mdc(a,b) = 1$
$\sqrt{2} = \frac{a}{b} \rightarrow a = b\sqrt{2} \rightarrow a^2 = 2b^2$
$b^2 \in \mathbb{Z}$
$\rightarrow a^2$ é par $\rightarrow a$ é par $\rightarrow a = 2k, k \in \mathbb{Z} \rightarrow (2k)^2 = ab^2$
$\rightarrow b^2 = 2k^2 \rightarrow b^2$ é par
$k^2 \in \mathbb{Z}$
$\rightarrow$ b é par $\rightarrow$ contradição. Portanto, $\sqrt{2}$ é irracional
